# 用户投资画像

## 基本信息
- **投资风格**: 趋势跟踪 + 事件驱动
- **风险偏好**: 中高风险，愿意使用杠杆（2 倍）
- **主要市场**: A 股、港股、美股
- **关注领域**: 科技股（存储芯片、云计算）

## 交易特征
- **反应速度**: 快，能够第一时间响应消息面
- **执行力**: 较强，能够知行合一
- **仓位管理**: 分仓操作，小量试仓 + 加仓策略
- **挂单策略**: 喜欢挂低价单等待回调

## 待验证假设
1. **存储芯片周期判断**: 需要验证存储行业上行周期的持续性和强度（海力士交易后优先级提升）
2. **云计算涨价传导**: 阿里云涨价是否能有效传导至业绩，以及市场反应强度（2026-03-18 已验证：市场反应强烈）
3. **高确定性事件的仓位策略**: 需要验证"第一时间满仓"vs"小量试仓 + 挂单"在不同确定性等级下的收益差异

## 认知盲区
1. **信念强度与资本约束的区分**: 需要分辨交易失误是源于认知不足还是资本限制（海力士交易暴露此问题）
2. **确定性事件的执行策略**: 高确定性机会下，如何平衡"第一时间买入"与"等待更好价格"的矛盾（阿里云交易证实此为盲区）
3. **挂单策略的场景适用性**: 趋势强劲时，挂低价等待回调可能错失机会，需要建立场景判断框架

## 情绪模式
- **错失恐惧**: 卖出后继续上涨会产生明显遗憾
- **反思倾向**: 能够主动复盘交易，分析得失
