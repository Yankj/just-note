# 财务自由之路 - 项目章程

**版本**：1.0  
**日期**：2026-03-20  
**状态**：已批准

---

## 一、项目愿景

**AI 原生·财务自由全周期成长教练**

不做基础记账，只服务有投资需求、追求长期财富增值的人群。

---

## 二、目标人群

| 维度 | 描述 |
|------|------|
| 年龄 | 25-45 岁 |
| 特征 | 有稳定收入、有投资行为、懂基础财商 |
| 痛点 | 投资无体系、交易心态差、无清晰规划 |

---

## 三、核心价值

1. 定制化财务自由路径规划
2. 投资行为 AI 复盘 + 交易心理诊断
3. 全资产可视化 + 被动收入追踪
4. 长期财富成长陪伴

---

## 四、6 大核心模块

| 模块 | 功能 |
|------|------|
| 财务自由规划 | 目标/路径/里程碑 |
| 投资日记复盘 | OCR/情感分析/逻辑评分 |
| 交易心理诊断 | 贪婪/恐惧识别/矫正 |
| 全资产视图 | 多平台聚合/收益率 |
| 被动收入追踪 | 覆盖率/预测 |
| AI 对话中枢 | 意图识别/任务分发 |

---

## 五、技术原则

1. **轻前端、重后端** — 业务逻辑在后端，前端只负责渲染
2. **代码清晰** — 任何代码都要易于理解和维护
3. **文档完整** — 每个决策都有文档记录
4. **可追溯** — 每次提交都有明确目的
