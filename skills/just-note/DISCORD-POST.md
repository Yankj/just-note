## 互动回复模板

### 如果有人问"和 flomo 比怎么样？"

```
好问题！主要区别：

just-note-ai 优势：
1. AI 自动分类 vs flomo 手动标签
2. 支持收支统计（flomo 没有）
3. 本地存储 vs 云端（数据自主）
4. 免费 vs ¥12/月

flomo 优势：
1. 更成熟稳定
2. 移动端体验更好
3. 更多用户

如果你重视数据自主和 AI 功能，选 just-note-ai

```

### 如果有人问"安全吗？数据会丢吗？"

```
好问题！

安全性：
✅ 数据存在你自己机器上（~/openclaw/workspace/memory/just-note/）
✅ 可以 Git 备份
✅ 可以导出为 flomo/Obsidian 格式
✅ 开源代码，透明可审计

不会丢：
✅ 本地文件，只要你不删就一直在
✅ 建议定期备份（Git/云盘）
✅ 支持导出，随时可以迁移

比云端更安全，因为数据在你手里 👍
```

### 如果有人问"怎么用？"

```
超简单！

1️⃣ 安装
npx clawhub install just-note-ai

2️⃣ 使用（两种方式）

方式 1：发消息（推荐）
微信/飞书发：「花了 200 块买书」
AI 自动分类保存

方式 2：CLI 命令
just-note write --type expense --amount 200 --content "买书"

3️⃣ 查看
just-note today  # 看今天的

就这么简单！试试？😊

文档：https://clawhub.ai/skills/just-note-ai
```
