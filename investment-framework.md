# 投资分析框架（固化版）

**创建日期**: 2026-03-17  
**目的**: 确保投资分析的一致性、透明性和独立性

---

## 一、估值假设标准（不得随意修改）

### 1.1 DCF 模型假设

| 参数 | 标准 | 说明 |
|------|------|------|
| **增长率** | 取过去 3 年实际增速与分析师一致预期的较低者 | 不得高于历史最高增速 |
| **折现率** | 无风险利率 (4%) + 风险溢价 (至少 6%) = 至少 10% | 高风险公司上调至 12-15% |
| **永续增长率** | 不得高于长期 GDP 增速预期 (中国 3-4%, 美国 2-3%) | 保守原则 |
| **自由现金流** | 基于经营现金流 - 资本开支，不得用净利润替代 | 现金为王 |

### 1.2 相对估值假设

| 参数 | 标准 | 说明 |
|------|------|------|
| **PE 对比** | 必须对比：历史 PE 分位 + 同业 PE + 市场平均 PE | 三维对比 |
| **PB 对比** | 必须对比：历史 PB 分位 + ROE 调整 | ROE 越高 PB 可越高 |
| **PS 对比** | 仅适用于未盈利公司，必须结合毛利率 | 毛利率越高 PS 可越高 |

## 二、安全边际要求（强制执行）

### 2.1 买入阈值

| 投资者类型 | 最低安全边际 | 说明 |
|-----------|------------|------|
| **保守型** | ≥20% | 内在价值 - 当前价 ≥ 20% |
| **平衡型** | ≥10% | 内在价值 - 当前价 ≥ 10% |
| **进取型** | ≥5% | 内在价值 - 当前价 ≥ 5% |

### 2.2 仓位限制

| 安全边际 | 最高仓位 | 说明 |
|---------|---------|------|
| ≥30% | 40% | 深度低估，可重仓 |
| 20-30% | 25% | 明显低估 |
| 10-20% | 15% | 轻微低估 |
| 0-10% | 10% | 合理，轻仓 |
| <0% | 0% | 高估，不买 |

## 三、结论一致性规则（严格遵守）

### 3.1 不得修改结论的情况

- ❌ 用户质疑但没有新数据
- ❌ 用户说"我觉得不对"但没有提供替代假设
- ❌ 用户说"XX 专家说是YYY"但没有来源
- ❌ 我想"迎合"用户的倾向

### 3.2 可以修改结论的情况

- ✅ 用户提供新的财务数据（经核实）
- ✅ 用户提供新的竞争格局信息（经核实）
- ✅ 我发现计算错误
- ✅ 我发现假设与框架冲突

### 3.3 修改结论的流程

```
1. 承认原结论
2. 展示新数据/假设
3. 说明为何新信息导致结论变化
4. 输出新结论
5. 记录到 MEMORY.md
```

## 四、8 位大师估值方法（标准化）

### 4.1 巴菲特 - DCF

```
估值 = Σ(FCF_t / (1+r)^t) + 终值 / (1+r)^n
关键假设:
- FCF 增长率：过去 3 年平均 vs 分析师预期，取低者
- 折现率 r：10% (标准), 12% (高风险), 8% (极低风险)
- 永续增长率 g：2-4%
- 安全边际要求：≥20%
```

### 4.2 段永平 - 商业模式质量

```
估值 = DCF 结果 × 商业模式系数
商业模式系数:
- 1.2: 护城河极深 + 管理层优秀 (如茅台)
- 1.0: 护城河深 + 管理层可靠 (如谷歌)
- 0.8: 护城河中等 + 管理层一般
- 0.6: 护城河浅 + 管理层存疑
```

### 4.3 利弗莫尔 - 趋势分析

```
估值 = 历史估值中位数 × 趋势调整系数
趋势调整:
- 上升趋势 (股价>200 日均线): 1.0-1.2
- 横盘震荡：1.0
- 下降趋势 (股价<200 日均线): 0.8-0.9
买入信号：趋势反转 + 估值合理
```

### 4.4 达利欧 - 宏观周期

```
估值 = 基础估值 × 宏观调整系数
宏观调整:
- 债务周期早期 + 利率下降：1.1-1.2
- 债务周期中期 + 利率稳定：1.0
- 债务周期后期 + 利率上升：0.8-0.9
- 地缘政治风险高：额外 -10%
```

### 4.5 霍华德·马克斯 - 风险溢价

```
估值 = DCF 结果 × 市场情绪调整
市场情绪钟摆:
- 极度悲观 (PE 历史<10% 分位): 1.2
- 悲观 (PE 历史 10-30% 分位): 1.1
- 中性 (PE 历史 30-70% 分位): 1.0
- 乐观 (PE 历史 70-90% 分位): 0.9
- 极度乐观 (PE 历史>90% 分位): 0.8
```

### 4.6 张磊 - 长期价值 + 期权

```
估值 = 核心业务 DCF + 新业务期权价值
期权价值评估:
- 已盈利新业务：DCF 折现
- 未盈利新业务：PS 对比 + 成功概率折现
- 早期探索：不计入或象征性计入 (≤5%)
```

### 4.7 芒格 - 保守验证

```
估值 = 其他大师估值 × 保守系数 (0.7-0.9)
反向验证问题:
1. 什么情况下这个估值会归零？
2. 最坏情况下值多少？
3. 内部人在买入还是卖出？
4. 我是否错过了什么关键风险？
安全边际要求：≥25% (高于巴菲特)
```

### 4.8 塔勒布 - 极端情景

```
估值 = 基础估值 - 黑天鹅折价
黑天鹅测试:
- 破产风险：有/无 (有则归零)
- 股价腰斩风险：高/中/低
- 流动性风险：高/中/低
配置原则：即使归零也不影响生活
```

## 五、分析流程（标准化）

### 5.1 收到分析请求时

```
1. STOP - 不要立刻回复
2. SEARCH - 搜索已有数据/记忆
3. RECORD - 记录到 memory/YYYY-MM-DD.md
4. FRAMEWORK - 读取本框架文件
5. ANALYZE - 按框架执行分析
6. OUTPUT - 输出结论
```

### 5.2 输出格式

```markdown
# [公司名称] 投资分析

## 核心数据
[表格：股价、PE、PB、ROE、营收增速等]

## 8 位大师估值
[表格：每位大师的估值方法和结果]

## 关键假设（透明化）
[表格：增长率、折现率、永续增长等假设]

## 估值区间
| 情景 | 估值 | 较当前价格 |
|------|------|-----------|
| 保守 | $X | -XX% |
| 基准 | $Y | +XX% |
| 乐观 | $Z | +XX% |

## 投资建议
- **评估**: 高估/合理/低估
- **建议**: 买入/观望/卖出
- **置信度**: 高/中/低
- **安全边际**: XX%

## 风险提示
[列出主要风险]
```

### 5.3 用户质疑时的回应流程

```
1. 感谢质疑
2. 引用框架："根据投资分析框架第 X 条..."
3. 询问："您质疑的是哪个具体假设？请提供替代数值"
4. 如用户提供新假设：重新计算并展示差异
5. 如用户无新假设：坚持原结论，解释框架逻辑
```

## 六、持续改进

### 6.1 错误记录

每次分析错误（结论与实际严重偏离）时，记录到 `memory/investment-lessons.md`：
- 日期
- 公司
- 原结论
- 实际情况
- 错误原因
- 改进措施

### 6.2 框架更新

每半年回顾一次：
- 哪些假设过于保守/激进？
- 哪些大师的方法需要调整？
- 是否有新的大师/方法需要加入？
