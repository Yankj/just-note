# 「记一下」Web App 产品设计文档

**版本**: v1.0  
**创建时间**: 2026-03-28  
**定位**: 零摩擦 AI 知识记录工具

---

## 一、产品定位

> **像发微信一样记录灵感，AI 自动帮你整理成知识库。**

### 目标用户

| 人群 | 特征 | 核心需求 |
|------|------|---------|
| 知识工作者 | 产品、运营、设计、内容创作者 | 快速记录灵感、整理思路 |
| 开发者/极客 | 技术背景、喜欢折腾 | 结构化记录、检索、导出 |
| 学生/研究者 | 学习、研究、写作 | 知识点整理、引用管理 |

### 核心价值

1. **零摩擦输入** - 无需选择分类、模板，直接输入
2. **AI 自动整理** - 自动分类、标签、关联
3. **智能检索** - 语义搜索、知识关联
4. **多端同步** - Web + 未来移动端

## 五、技术架构

### 5.1 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端框架** | React 18 + TypeScript | 类型安全、组件化 |
| **构建工具** | Vite | 快速开发、热更新 |
| **UI 框架** | Tailwind CSS + Headless UI | 原子化 CSS、无样式组件 |
| **状态管理** | Zustand | 轻量、简洁 |
| **路由** | React Router v6 | 声明式路由 |
| **后端** | Node.js + FastAPI | 快速开发 |
| **数据库** | SQLite (MVP) → PostgreSQL | 简单 → 扩展 |
| **AI 处理** | OpenClaw Gateway | 复用现有能力 |
| **部署** | Vercel (前端) + 云服务器 (后端) | 快速上线 |

### 5.3 数据模型

```typescript
interface Note {
  id: string;
  type: NoteType;  // inspiration|idea|knowledge|expense|income|diary|task|quote|other
  title: string;
  content: string;
  tags: string[];
  amount?: number;  // 收支金额
  currency?: string;  // CNY|USD|...
  createdAt: string;  // ISO 8601
  updatedAt: string;
  dayId: string;  // YYYY-MM-DD
  relatedIds?: string[];
  aiSummary?: string;
}

interface NoteType {
  id: string;
  name: string;
  icon: string;
  color: string;
}

interface UserSettings {
  theme: 'light' | 'dark' | 'auto';
  defaultType: NoteType;
  autoTag: boolean;
  notifications: boolean;
}
```
